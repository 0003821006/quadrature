The Accelerate framework developed by Apple Inc. provides many APIs, including those for linear algebra (BLAS and LAPACK) and quadrature.
You can figure out how to use Accelerate's BLAS and LAPACK by following the references in Accelerate.h.
However, using the quadrature API was difficult.
Fortunately, a sample program for quadrature can be found at https://face2d.com/apple/accelerate/Quadrature/-/tree/master.
